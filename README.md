# MOZUMDER PHONE CENTER - Premium Invoice Generator

A modern, high-end automated billing system built with Next.js and pure CSS. This application allows you to generate, preview, and print beautiful invoices with a premium aesthetic and gold/silver metallic accents.

## Features
- **Live Preview:** See your changes instantly as you type.
- **Premium Design:** High-end typography (Montserrat) and custom metallic CSS gradients matching the MOZUMDER brand.
- **Print Optimization:** Designed to perfectly fit on a single A4 page. Uses `@media print` to strip away the editor UI and format the invoice flawlessly for saving as a PDF or printing.
- **Dynamic Calculations:** Automatically computes line item amounts, subtotals, configurable taxes, and grand totals.

## Technologies Used
- Next.js (App Router)
- React
- Vanilla CSS
- Lucide React (Icons)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/fahid247/Fahim-invoice.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to use the invoice generator.
